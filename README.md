# POC vanilla js boilerplate

## Installation

The boilerplate requires :

- [Node.js](https://nodejs.org/) v18+
- [PNPM](https://pnpm.io/)

To get started, just run

```sh
  nvm use
  pnpm install
  pnpm dev
```
